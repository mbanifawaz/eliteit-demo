IMAGE PLACEHOLDERS

This folder contains placeholder image files that should be replaced with high-quality images for the live website. 

Recommended image sizes:
- hero-image.jpg: 1200×800px
- about-image.jpg: 800×600px
- service images (erp-service.jpg, web-service.jpg, etc.): 600×400px

For optimal performance, compress all images before using them on the website.

Suggested free image sources:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)
